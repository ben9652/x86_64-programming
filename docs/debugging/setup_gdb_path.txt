# Cómo agregar GDB permanentemente al PATH de Windows

## Opción 1: Via PowerShell (como administrador)
[Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\msys64\mingw64\bin", [EnvironmentVariableTarget]::Machine)

## Opción 2: Via GUI de Windows
1. Presiona Win + X y selecciona "Sistema"
2. Haz clic en "Configuración avanzada del sistema"
3. Haz clic en "Variables de entorno"
4. En "Variables del sistema", selecciona "Path" y haz clic en "Editar"
5. Haz clic en "Nuevo" y agrega: C:\msys64\mingw64\bin
6. Haz clic en "Aceptar" en todas las ventanas

## Opción 3: Via línea de comandos (cmd como administrador)
setx PATH "%PATH%;C:\msys64\mingw64\bin" /M

## Verificar que funciona:
# Cierra y abre una nueva ventana de PowerShell, luego ejecuta:
gdb --version
